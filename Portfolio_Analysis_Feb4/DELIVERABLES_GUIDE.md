## Introduction

This guide provides an overview of all strategic deliverables produced in support of Kevin McCarron's business portfolio. ## Mobile Clinic by OnRequest

### Deliverable 1: RHTP Funding Capture Package

**Location:** `/deliverables/mobile_clinic/rhtp_funding_package/`

**Purpose:** To position Mobile Clinic by OnRequest as the ideal partner for Kentucky's $212.9 million Rural Health Transformation Program (RHTP) and to provide a ready-to-submit funding proposal.

**Audience:** Kevin McCarron (internal review), Kentucky Cabinet for Health and Family Services (CHFS), Kentucky Department for Public Health (DPH), and other relevant state agencies.

**Utilization:** This package should be used to proactively engage with state decision-makers and to submit a formal funding proposal as soon as the application window opens in early 2026. The Executive Summary Brief is designed for high-level stakeholder conversations, while the Funding Landscape Analysis and Positioning Document provide the depth required for a formal application.

**Contents:**
| Document | Description |
| :--- | :--- |
| `executive_summary_brief.md` | A 1-page summary for decision-makers, highlighting the opportunity and recommended next steps. |
| `kentucky_rhtp_funding_landscape_analysis.md` | A detailed analysis of the RHTP program, including priorities, timelines, and key stakeholders. |
| `positioning_document.md` | A formal positioning document that articulates Mobile Clinic's alignment with RHTP priorities and its proven delivery capability. |

| Document | Description |

| `hpsa_coverage_gap_analysis.md` | An analysis of Health Professional Shortage Area (HPSA) designations in Central Kentucky, identifying counties with the most critical need. |
